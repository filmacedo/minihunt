# MiniHunt Design System

## Colors
- Black: `#000000` (background)
- Neutral 800: `#262626` (cards)
- Neutral 400: `#A3A3A3` (borders)
- Neutral 200: `#E5E5E5` (secondary text)
- Neutral 100: `#F5F5F5` (primary text)
- Hi-Viz Yellow: `#E1FF00` (critical CTAs only - Bet/Claim buttons)

## Typography
- **Geist**: All text
- **Geist Mono**: All numbers (prices, counts, dates, rankings)
- **Weights**: Regular (400), Semibold (600) only

## Component Specs
- Cards: 72px height, Neutral 800 bg, 12px radius
- Buttons (Primary): Hi-Viz bg, black text, 8px radius
- Buttons (Secondary): Neutral 800 bg, white text, 1px border
- Modals: Neutral 800 bg, 400px max-width, 16px radius
- FAB: 56x56px, Hi-Viz bg, bottom-6 right-6
- Tap targets: 44px minimum
